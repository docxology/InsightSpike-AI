## 推奨戦略

### 案1: Transformer版を優先（推奨）

1. **JSAI2026**: Option B（Transformer）を投稿
2. **arxiv**: フル版（迷路+RAG+Transformer）を同時期に公開
3. **次の学会**: Option A（迷路）を理論寄りの会議に投稿

**理由**:
- Transformerは注目度が高い
- 因果検証（F正則化）がある
- 迷路はarxivで詳細に書ける

### 案2: 両方出す

- Option A → 知識表現セッション
- Option B → 深層学習セッション

**リスク**: 審査員の重複、メッセージの分散

### 案3: 迷路版を優先

- geDIGの理論的基盤を先に確立
- Transformerは「応用」として次に回す

**リスク**: インパクトが弱く見える
