## Publishing Checklist

### Before Publishing

- [ ] Review for accuracy
- [ ] Test all code examples
- [ ] Add images/diagrams if helpful
- [ ] Check links work

### Zenn

1. Create new article on [zenn.dev](https://zenn.dev)
2. Copy content from `zenn_gedig_intro.md`
3. Set `published: true` in frontmatter
4. Review preview
5. Publish

### Qiita

1. Create new article on [qiita.com](https://qiita.com)
2. Copy content (remove Zenn-specific frontmatter)
3. Add appropriate tags
4. Publish

### Medium

1. Create new story on [medium.com](https://medium.com)
2. Copy content from `medium_gedig_intro.md`
3. Add images
4. Submit to relevant publications (Towards Data Science, etc.)

### dev.to

1. Create new post on [dev.to](https://dev.to)
2. Copy content
3. Add frontmatter: `title`, `tags`, `cover_image`
4. Publish
