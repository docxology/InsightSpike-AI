# 普遍原理仮説

> **仮説**: 構造コストと情報利得のバランス（F = Δ構造 − λ·Δ情報）は、すべての知的・適応的システムに共通する根本的な設計原理である。

## 核心となるアイデア

AIが知識グラフをいつ更新すべきかを制御する同じ方程式が、以下も記述できるとしたら：

- 脳が何を記憶するかの判断
- 細胞がいつ分裂するかの判断
- 植物がどこで分岐するかの判断
- 組織がいつ変化するかの判断

geDIGのゲージ：

```
F = ΔEPC_norm − λ·(ΔH_norm + γ·ΔSP_rel)
```

を抽象化すると：

```
F = 構造コスト − λ × 情報利得
```

これは数学的等価性の主張ではなく、**設計原理**である：この2つの量のバランスをとるシステムは、適応的で知的な振る舞いを示すかもしれない。

## なぜ普遍的かもしれないのか

### 根本的なトレードオフ

成長し、学習し、適応するすべてのシステムは、同じジレンマに直面する：

| 変化しすぎると... | 変化しなさすぎると... |
|------------------|---------------------|
| 構造が崩壊する | 構造が硬直する |
| アイデンティティが失われる | 適応に失敗する |
| ノイズが信号を圧倒する | 機会を逃す |

「スイートスポット」は**構造コスト ≈ 情報利得**の場所。

### スケールを超えた現れ方

| システム | 構造 | 情報 | バランス |
|---------|------|------|---------|
| **ニューロン** | シナプス重み | 予測誤差 | ヘブ則 + 恒常性 |
| **脳** | コネクトーム | 感覚的驚き | 睡眠中の統合 |
| **細胞** | DNA/代謝 | 環境シグナル | 恒常性 |
| **植物** | 枝の構造 | 光/栄養勾配 | フラクタル成長 |
| **生物** | ボディプラン | 生存フィードバック | 進化 |
| **組織** | プロセス/階層 | 市場シグナル | イノベーションサイクル |
| **AIシステム** | 知識グラフ | クエリ結果 | **geDIG** |

## 既存理論との接続

### 自由エネルギー原理（FEP）

Karl Fristonの FEP は、生物システムが変分自由エネルギーを最小化すると述べる：

```
F_FEP = 複雑さ + 不正確さ
      = D_KL(q||p) + E_q[-log p(o|s)]
```

**接続**: geDIGの構造コストは「複雑さ」に、情報利得は「（負の）不正確さ」に対応する。

### 最小記述長（MDL）

MDL原理：最良のモデルは総記述長を最小化する：

```
L = L(モデル) + L(データ|モデル)
```

**接続**: ΔEPC ∝ ΔL(モデル), ΔIG ∝ −ΔL(データ|モデル)

### 熱力学

ヘルムホルツ自由エネルギー：

```
F = U − TS
```

**接続**: 構造コスト → 内部エネルギー (U), 情報利得 → エントロピー項 (TS), λ → 温度 (T)

## 内発報酬という特性

重要な特徴：**Fは完全に内部状態から計算される**

| 外発報酬 | 内発報酬（F） |
|---------|-------------|
| オラクルが必要 | 自己計算 |
| タスク固有 | タスク非依存 |
| ハック可能 | 構造に紐づく |
| 設計が必要 | 原理から創発 |

これが意味すること：
- 「何が良いか」を指定する必要がない
- システムが自分の進歩の概念を定義する
- **自律性**と**恒常性**が自然に創発する

## 検証可能な予測

この仮説が正しければ：

### 予測1: 安定点でF ≈ 0
適応的システムは安定期間中、F = 0（バランス）付近で動作するはず。

**検証**: 訓練済みニューラルネット、健康な細胞、成熟した植物でFを測定。

### 予測2: 変化の前にFがスパイク
大きな遷移の前に、Fは強く正または負になるはず。

**検証**: 学習のブレイクスルー、細胞分裂、枝の形成中にFを追跡。

### 予測3: F最適化が上手い = 適応が上手い
Fをより効果的に最小化するシステムは、より良く汎化するはず。

**検証**: F正則化ありとなしの訓練をニューラルネットで比較。

### 予測4: 人間の「ひらめき」= Fの急落
主観的な「アハ体験」は、突然のF減少と相関するはず。

**検証**: 問題解決中のEEG/fMRI + F推定。

## 足りないもの

1. **因果的証拠**: 相関 ≠ 因果。介入実験が必要。

2. **生物学的検証**: 理論はニューロン/細胞がこれに従うことを示唆するが、測定はない。

3. **数学的必然性**: なぜこの形式か？他の形式でも同様に機能するか？

4. **スケール不変性**: λはスケールを超えて変化するか？どのように？

5. **失敗モード**: この原理はいつ破綻するか？
