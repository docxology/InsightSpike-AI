# 🎮 InsightSpike ローカルアプリの遊び方

このガイドでは、セットアップから実際に知識を育てて遊ぶまでの手順を紹介します。

## 1. 準備 (Ollama)

このアプリはローカルLLM（AIの頭脳）として **Ollama** を使用します。
まだインストールしていない場合は、以下の手順で準備してください。

1. [Ollama公式サイト](https://ollama.com/) からダウンロードしてインストール。
2. ターミナルで以下のコマンドを実行して、モデル（Mistral推奨）をダウンロード・起動します。

```bash
ollama serve
# 新しいターミナルタブで実行
ollama pull mistral
```

## 2. アプリの起動

InsightSpikeのディレクトリで、以下のコマンドを実行するだけでアプリが立ち上がります。

```bash
./run_local_app.sh
```

ブラウザが自動的に開き、アプリ画面（黒基調のダークモード）が表示されます。

## 3. まずは接続

1. 左側のサイドバーにある **"⚙️ Connection Settings"** を確認します。
   - Provider: `ollama`
   - Model: `mistral`
   - API Base: `http://localhost:11434/v1`
2. **"🚀 Connect / Start Engine"** ボタンを押します。
3. "🟢 Connected: mistral" と表示されれば準備完了です！

## 4. 遊び方シナリオ

### シナリオA: 基本的な会話
まずは普通のチャットボットとして話しかけてみましょう。
- **Chatタブ** の下部入力欄に「こんにちは、自己紹介してください」と打ってみましょう。
- Mistralが応答してくれます。

### シナリオB: 知識を教える（Ingest）
ここからが本番です。AIに新しい知識を教え込み、グラフを作らせましょう。

1. **"📥 Ingest" (学習)** タブを開きます。
2. テキストエリアに、覚えさせたい文章を入力します。
   - 例: `examples/sample_knowledge.txt` の中身をコピー＆ペーストしてみましょう。
   - あるいは、自分の趣味（例：「コーヒーの美味しい淹れ方」）について書いてみましょう。
3. **"🧠 Learn & Memorize"** ボタンを押します。
4. "Knowledge successfully integrated!" と出れば、AIが知識を構造化して保存しました。

### シナリオC: 脳の中身を覗く（Graph）
教えた知識がどうなったか見てみましょう。

1. **"🕸️ Knowledge Graph"** タブを開きます。
2. **"🔄 Refresh Visualization"** ボタンを押します。
3. 画面にノード（丸）とエッジ（線）のネットワークが表示されます！
   - ノードをマウスで掴んでぐりぐり動かせます。
   - 「InsightSpike」や「geDIG」といった単語がつながっているのが見えるはずです。

### シナリオD: 育てた知識を使う
もう一度チャットに戻りましょう。

1. **"💬 Chat"** タブに戻ります。
2. さっき教えた内容について質問します。
   - 例：「InsightSpikeの主な機能は何？」
3. AIは、あなたが教えた知識グラフを参照して回答します。

## 5. 終了と再開
アプリを閉じても、知識は `data/local_app` に保存されています。
次に `./run_local_app.sh` で起動したときも、育てたグラフはそのまま残っています。どんどん知識を大きくしていきましょう！
