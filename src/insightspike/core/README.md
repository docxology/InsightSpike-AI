# Core Module

Base classes and generic interfaces for the InsightSpike agent system.

## Submodules

| Module | Purpose |
|--------|---------|
| `base/` | Generic interfaces, data models |
| `interfaces/` | Abstract agent interfaces |
| `protocols/` | Protocol definitions |

## Key Classes

- `EnvironmentInterface` — Abstract environment interface
- `InsightMoment` — Insight event representation
- `TaskType` — Task type enumeration
