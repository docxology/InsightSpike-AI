# Algorithms Module

Core algorithms including geDIG (Graph Edit Distance for Insight Gauge).

## Key Components

| Algorithm | Purpose |
|-----------|---------|
| `gedig_calculator` | Graph Edit Distance calculations |
| `gedig_ab_logger` | A/B logging for geDIG |
| `graph_metrics` | Graph structure metrics |

## geDIG Model

The geDIG (Graph Edit Distance as Insight Gauge) model formalizes 'insight' as discovering minimal graph transformations for structural isomorphism.
