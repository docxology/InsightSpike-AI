# Output Directory

Generated outputs from docxology executions.

## Structure

```
output/
├── analysis/        # Analysis results
├── animations/      # Animated visualizations
├── figures/         # Static plots
│   ├── advanced/    # Advanced visualizations
│   └── extended/    # Extended analysis plots
├── logs/            # Execution logs
├── reports/         # Generated reports
│   ├── comprehensive_report_*/

```

## File Formats

- **JSON** — Structured data with metadata
- **PNG** — Plot images
- **HTML** — Interactive reports
- **Markdown** — Text summaries
- **GIF/MP4** — Animations
