# Example Scripts

Usage demonstrations for the docxology framework.

## Available Scripts

| Script | Description |
|--------|-------------|
| `run_all_methods.py` | Execute all discovered methods |
| `generate_report.py` | Generate comprehensive reports |
| `analyze_results.py` | Analyze execution results |
| `extended_analysis.py` | Advanced statistical analysis |
| `generate_all_visualizations.py` | Generate all visualization types |

## Quick Start

```bash
# Run all methods
python examples/run_all_methods.py

# Generate report
python examples/generate_report.py

# Generate visualizations
python examples/generate_all_visualizations.py
```

## Output

All scripts output to `output/` directory:
- `output/reports/` — Generated reports
- `output/figures/` — Visualizations
- `output/logs/` — Execution logs
